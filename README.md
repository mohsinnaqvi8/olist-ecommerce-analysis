# E-commerce Data Analysis Project (Olist Dataset)

## Overview
Performed end-to-end data analysis on a real-world e-commerce dataset (100K+ orders) using SQL (DuckDB).

## Key Analysis Areas
- Order trends (monthly, weekday, hourly)
- Revenue analysis (~$15.8M total)
- Customer behavior (repeat vs one-time)
- Product category performance
- Delivery time and delay analysis

## Key Insights
- Strong upward growth in orders and revenue
- Peak activity during weekdays and mid-day hours
- Majority customers are one-time buyers
- Low delivery delay (~6.7%) indicates efficient logistics

## Tools Used
- SQL (DuckDB)
